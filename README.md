### 安装方法
#### NPM安装
```shell
npm i hestudio-bing-wallpaper-get
node node_modules/hestudio-bing-wallpaper-get/get.js
```

#### 手动安装
```shell
git clone https://gitlab.com/heStudio/bing-wallpaper-get.git
cd bing-wallpaper-get
npm run server
```
